## Install and Run

#### 1) First go to ````./assets/.env```` AND ````./assets/config/config.json```` and replace your data
#### 2) Then run the install command
```shell
yarn install && yarn run dev
```
## `Or` if you use npm
```shell
npm i && npm run dev
```

### 3) Success!
